Schedular

A simple POST API request can be used to create a FaaS instance. The JSON data structure contains the parameters like payload source URL, programming language and datasource destination.

Deploy your code

The code that needs to run on the FaaS instance can be downloaded from anywhere. And the result can also be uploaded anywhere.

Programming Language Suppport

In general, any binary compiled for Linux or FreeBSD. Besides running binaries we also provide FaaS instances with the SDK included of the required programming language:

  • .NET (6.0/7.0)
  • Go
  • PHP8
  • Python3
  • Rust
  • Just ask, and we can provide the required SDK FaaS environment for you.
